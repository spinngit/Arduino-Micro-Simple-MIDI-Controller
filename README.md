# ArduinoMicroMIDIController

This is a simple USB MIDI controller with 1 Pitch Fader, 1 Toggle Switch, 1LED and Arduino Micro.

- MIDIUSB library is required.
https://github.com/arduino-libraries/MIDIUSB
